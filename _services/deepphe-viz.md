DeepPhe results that are stored in a SQLite database such as those created by DeepPhe Translational
can be utilized with the 
[DeepPhe Visualizer](https://github.com/DeepPhe/DeepPhe-Visualizer-v2),
which provides a web-based interface to data, supporting
review of both cohorts and individual patient records. 

See the [DeepPhe Visualizer 2 User Guide](https://deepphe.github.io/DeepPhe-Visualizer-v2/)
for installation and usage documentation.

Documentation: [DeepPhe Data API](https://deepphe.github.io/dphe-data-api/) for the API layer that serves the database read by DeepPhe Visualizer.

Supported by the [National Cancer Institute's](https://www.cancer.gov)
[Information Technology for Cancer Research (ITCR) initiative](https://itcr.cancer.gov) 
(Grant #[U24CA248010](https://reporter.nih.gov/project-details/11231451)).
